What is coupled oscillator?

A coupled oscillator are set of oscillators connected through an energy transfering components. The most basic example of a coupled oscillator is two spring oscillators connected via a third spring. Free body diagram

Before we start creating free body diagrams and write out equation of motion for each masses, we need to clear out few assumptions.
  • Both masses $m$ are equal
  • All the three springs have same stiffness i.e. $k$
  • Left wall lies on $x = 0$ coordinate
Now looking at block 1, we can assume a slight displacement $x_1$ towards right. This causes spring 1 to extend. Similarly for block 2, we can assume a slight displacement $x_2$ towards right. This causes spring 3 to compress. For spring 2, it will pull block 1 with $k(x_2-x_1)$ and block 2 with $-k(x_2-x_1)$. Keeping displacement terms in order: $-k(x_1-x_2)$ and block 2 with $k(x_1-x_2)$. Equating forces on each masses we get following equations of motion: $$-kx_1 - k(x_1 - x_2) = m\ddot{x_1} \tag{1}$$ $$-kx_2 + k(x_1 - x_2) = m\ddot{x_2} \tag{2}$$

Solving Equations of Motion

To solve equations $1$ and $2$ we can use a trick. We can add and subtract both equations to get a new set of equations which are equivalent to original. $$-kx_2 + k(x_1 - x_2) + kx_1 + k(x_1 - x_2) = m(\ddot{x_2} - \ddot{x_1})\tag{2 - 1}$$ $$-3k(x_1 - x_2) = m(\ddot{x_1} - \ddot{x_2})\tag{2 - 1}$$ $$-kx_2 + k(x_1 - x_2) - kx_1 - k(x_1 - x_2) = m(\ddot{x_2} + \ddot{x_1})\tag{2 + 1}$$ $$-k(x_1 + x_2) = m(\ddot{x_1} - \ddot{x_2})\tag{2 + 1}$$ We can see that there are two equation in form of $(x_1 - x_2)$ and $(x_1 + x_2)$. Naming them $x_{+} = x_1 + x_2$ and $x_{-} = x_1 - x_2$. We get
$$-kx_{+} = m\ddot{x}_{+}$$ $$-3kx_{-} = m\ddot{x}_{-}$$
